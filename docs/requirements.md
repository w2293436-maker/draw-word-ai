# 画词AI · Draw Word — 项目需求文档

## 项目概述
一款面向各年龄段英语学习者的 AI 漫画生成工具。用户提交英语短文，AI 自动拆分为漫画分镜并生成插图，辅以词汇解析、语法讲解和语音朗读，让单词记忆不再枯燥。

## 目标用户
- 中小学生（课本短文 → 漫画辅助理解）
- 大学生/成人（新闻/文章 → 趣味化学习）
- 英语教师（教学素材制作）

## 核心功能

### V1.0（已完成）
| 功能 | 描述 | 状态 |
|------|------|------|
| 文本输入 | 支持最多5000字符英语短文 | ✅ |
| AI 分镜 | DeepSeek 自动拆分3-6个漫画分镜 | ✅ |
| AI 生图 | 通义万相生成漫画插图（3种风格） | ✅ |
| 词汇提取 | 每句2-4个重点词汇，带释义和例句 | ✅ |
| 语法讲解 | 逐句语法点 + 全局语法总结 | ✅ |
| 内容审核 | 关键词 + AI 双重过滤不良内容 | ✅ |
| 可视化设置 | 用户可自定义颜色、字号、圆角等 | ✅ |
| 响应式设计 | PC + 手机自适应 | ✅ |

### V1.1（规划中）
| 功能 | 描述 |
|------|------|
| 语音朗读 | TTS 朗读英语原文 |
| 图片下载 | 一键保存漫画拼接长图 |
| 学习记录 | 历史生成记录 + 收藏 |

### V2.0（远期）
| 功能 | 描述 |
|------|------|
| 用户系统 | 登录注册、个人词库 |
| 生词本 | 自动积累生词，间隔复习 |
| 练习题 | 根据短文自动生成选择题/填空题 |
| 多语言 | 支持日语、韩语等其他语言 |

## 非功能需求
- 响应时间：从提交到出图 ≤ 30秒
- 安全性：色情/暴力/恐怖内容 100% 拦截
- 兼容性：Chrome / Safari / Edge / 手机浏览器
- 成本控制：单次生成 ≤ ¥1
