# 画词AI · Draw Word — 内容安全规范

## 审核机制（四道防线）

```
用户输入
  → ① 关键词过滤（200+ 中英文敏感词）
  → ② DeepSeek AI 深度审核
  → ③ 通义万相 negative_prompt
  → ④ 阿里云平台合规审查
```

## 拦截类别

### 色情/性暗示
- 直接色情词汇（中英文）
- 性暗示描写（身体部位挑逗、暧昧场景）
- 低俗擦边球（透视装、湿身、暴露、内衣描写等）
- 不恰当的身体接触描写

### 暴力/血腥
- 杀害、虐待、肢解等暴力行为
- 血腥场面描写
- 恐怖/惊悚内容

### 其他违规
- 毒品相关内容
- 自杀/自残描写
- 违法/政治敏感内容

## 审核原则
- **宁可误拦，不可放过**：面向青少年学习者，安全第一
- **上下文判断**：正常运动出汗 ✅  vs  性暗示湿身描写 ❌
- **组合判断**：单独词汇可能无害，但组合产生暗示 → 拦截

## 敏感词库位置
`server/services/ai.js` → `BLOCKED_SEXUAL` / `BLOCKED_VIOLENCE` / `BLOCKED_OTHER`
