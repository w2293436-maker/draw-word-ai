# 一、项目产品定位（核心）

## 1. 项目核心用途、解决的用户痛点

**画词AI**是一款面向外语学习者的 AI 漫画生成工具。用户提交一段英语或西班牙语短文，系统自动将文本拆分为漫画分镜，生成插图，并配套提供词汇解析、语法讲解和语音朗读。

**解决的核心痛点：**

| 痛点 | 传统方式 | 画词AI |
|------|------|------|
| 单词记忆枯燥 | 死记硬背单词书 | 短文场景中自然习得，视觉化记忆 |
| 阅读理解困难 | 逐句查字典 | 漫画图解 + 逐句翻译 + 语法讲解 |
| 听力练习断层 | 找音频资源麻烦 | 一键逐句朗读（英语/西语） |
| 学习素材匮乏 | 课本内容单一 | 任何短文都能变教材 |
| 学习动力不足 | 缺乏正反馈 | 生成漫画的成就感 + 收藏生词 |

## 2. 目标使用人群、使用场景

| 人群 | 典型场景 |
|------|------|
| 🧒 中小学生 | 课本短文 → 漫画辅助理解 + 记单词 |
| 👨‍🎓 大学生/成人 | 新闻/文章 → 趣味化精读 + 生词积累 |
| 👩‍🏫 外语教师 | 备课素材制作、课堂展示 |
| 👶 低龄启蒙（幼儿） | 简单绘本故事，绘本风格漫画 |
| 🌍 多语言学习者 | 英语 + 西班牙语，后续可扩展 |

**使用环境：** 手机/平板/电脑浏览器均可访问，无需安装 App。

## 3. 项目核心价值、差异化优势

| 维度 | 竞品（传统工具） | 画词AI |
|------|------|------|
| 学习方式 | 文字为主 | **漫画视觉化** — 右脑图像记忆 |
| 内容来源 | 固定教材 | **任意短文**均可生成 |
| 学习深度 | 单一查词 | 词汇 + 语法 + 朗读 + 漫画 **四合一** |
| 使用门槛 | 需下载App/付费 | **浏览器打开即用**，免费额度 |
| 语言支持 | 单一语言 | **英西双语**，切换即用 |
| 安全性 | 无审核 | **四道防线**审核，青少年友好 |

## 4. 项目最终交付的能力与落地意义

**交付能力：** 输入短文 → AI 自动生成漫画分镜 + 词汇表 + 语法总结 + 逐句朗读 → 可收藏生词、回看历史、分享成果

**落地意义：** 把"枯燥的外语学习"变成"看漫画学语言"，降低学习门槛，提升学习动力。特别适合碎片化时间（等车、排队时看几格漫画学几个单词）。

## 2. 整体运行逻辑框架（通俗说明）

```
用户打开网页
  → 同意使用协议（首次）
  → 粘贴英语/西语短文
  → 选择漫画风格 + 语言
  → 点击"生成漫画"
  → AI阅读文本，拆成3-6个画面
  → AI为每个画面生成插图
  → 同时提取词汇、分析语法
  → 展示漫画 + 词汇表 + 语法
  → 用户可以：逐句听朗读 / 收藏单词 / 分享 / 回看历史
```

整个过程约10-20秒。

| 生词本 | 学习中遇到的生词一键收藏，后续复习 |

| 数据看板 | 运营者查看使用情况、用户增长、成本控制 |
| 安全审核 | 面向青少年，确保内容干净安全 |
| 个性化设置 | 换颜色/调字号，让界面更符合个人喜好 |
| 分享 | 觉得漫画有趣，分享给朋友/同学 |
| 合规 | 法律风险规避，用户权益保护 |

## 4. 整体产品运行前提、使用条件

**运行前提：**
- 浏览器能访问网址（PC/手机均可）
- 服务端 AI API 正常连接（DeepSeek + 通义万相）
- Python 环境可用（语音朗读功能依赖）

**使用限制：**
- 单次输入上限：5000字符
- 单次生成耗时：约10-20秒
- 历史记录：保留最近20条
- 生词本：存储在浏览器本地，清除缓存会丢失
- 免费额度：取决于 API 账户余额

# 四、核心功能详细拆解（产品视角）

## 功能1：短文输入与语言/风格选择

- **功能目的：** 让用户提供学习素材并指定生成偏好
- **触发方式：** 手动输入/粘贴
- **适用场景：** 用户有一段想学习的外语短文
- **输入要求：** 文本长度10-5000字符，英语或西班牙语
- **产出结果：** 系统接收文本准备进行AI处理
- **功能边界：** 不支持中文、日语等其他语言（v1.5仅英语/西语）；不支持图片/PDF上传

## 功能2：AI漫画生成

- **功能目的：** 将短文自动拆分为3-6格漫画，每格配插图
- **触发方式：** 点击"生成漫画"按钮
- **适用场景：** 用户提交短文后，想看漫画版的文章内容
- **输入要求：** 已填写短文 + 已选语言 + 已选风格
- **产出结果：** 3-6格漫画分镜，每格包含插图+原文+翻译+词汇+语法
- **功能边界：** 图片风格可能不完全一致（AI生图局限性）；生成耗时10-20秒；图片质量受AI模型限制

## 功能3：词汇提取与学习

- **功能目的：** 自动从文中提取重点词汇，提供释义和例句
- **触发方式：** 自动（漫画生成时同步完成）
- **适用场景：** 用户想积累生词，系统自动提取节省查词典时间
- **输入要求：** 无（全自动）
- **产出结果：** 每句2-4个重点词汇，含中文释义和英文例句
- **功能边界：** AI提取的词汇可能不是用户想要的；数量固定2-4个

## 功能4：语法讲解

- **功能目的：** 解析文中的核心语法知识点
- **触发方式：** 自动生成+手动点击查看
- **适用场景：** 用户想了解文中的语法规律，辅助理解
- **输入要求：** 无（全自动）
- **产出结果：** 每句1个语法点+全局3-5条语法总结
- **功能边界：** 语法解析为AI生成，可能存在不准确情况

## 功能5：逐句语音朗读

- **功能目的：** 用纯正发音朗读每句，辅助听力练习
- **触发方式：** 手动点击每句旁的🔊按钮
- **适用场景：** 用户想听某句的发音；跟读练习
- **输入要求：** 已生成漫画
- **产出结果：** 播放该句的英语/西语发音
- **功能边界：** 部分旧手机浏览器不支持（此时使用服务端音频）；全文朗读点顶部"听朗读"按钮

## 功能6：生词本

- **功能目的：** 收藏学习过程中遇到的生词，集中管理和导出
- **触发方式：** 手动点击漫画中的单词标签
- **适用场景：** 用户想积累生词，后续复习或导入其他工具
- **输入要求：** 已生成漫画
- **产出结果：** 生词列表，支持导出CSV文件（可用Excel打开）
- **功能边界：** 数据存于浏览器本地，清除缓存会丢失；无云同步

## 功能7：历史记录

- **功能目的：** 自动保存每次生成的漫画，支持回看
- **触发方式：** 自动保存+手动查看
- **适用场景：** 用户想回看之前学过的内容
- **输入要求：** 至少生成过一次漫画
- **产出结果：** 最近20条生成记录，点击即可重新加载
- **功能边界：** 仅保留最近20条；数据存浏览器本地

## 功能8：数据看板

- **功能目的：** 提供运营数据分析，了解使用情况和用户行为
- **触发方式：** 手动访问 `/dashboard.html`，输入密码
- **适用场景：** 运营者/产品经理查看使用数据和趋势
- **输入要求：** 密码（默认 drawword2026）
- **产出结果：** KPI卡片+每日趋势图+语言风格分布+周月环比+用户分层+留存率
- **功能边界：** 需密码访问；数据为匿名统计，不含个人信息

## 功能9：内容安全审核

- **功能目的：** 拦截色情、暴力、恐怖等不良内容
- **触发方式：** 自动（每次生成时）
- **适用场景：** 所有生成请求，保护青少年用户
- **输入要求：** 无
- **产出结果：** 违规内容被拦截并提示
- **功能边界：** AI审核可能漏判；极端擦边内容可能无法100%拦截

## 功能10：分享

- **功能目的：** 将生成的漫画摘要分享给他人
- **触发方式：** 手动点击"📤 分享"按钮
- **适用场景：** 用户觉得漫画有趣，想分享给朋友
- **输入要求：** 已生成漫画
- **产出结果：** 手机调起原生分享面板 / 电脑复制漫画摘要到剪贴板
- **功能边界：** 分享的是文字摘要，不是漫画图片；无法分享完整漫画图

## 功能11：个性化设置

- **功能目的：** 用户自定义界面颜色、字号、圆角、标题
- **触发方式：** 原可通过设置按钮进入（当前已隐藏按钮，功能代码保留）
- **适用场景：** 用户想换颜色/调字号
- **产出结果：** 所有设置实时预览，保存后持久生效
- **功能边界：** 设置按钮已隐藏（如有需要可恢复）

# 五、业务分支&场景化逻辑

## 1. 常规正常使用场景

**场景A：学生课后学习**
英语课本上一篇短文 → 粘贴到画词AI → 选英语+绘本风格 → 生成漫画 → 边看漫画边学词汇 → 收藏生词 → 导出CSV复习

**场景B：成人自学西语**
西语新闻一段 → 切换到西班牙语 → 选日式漫画 → 生成漫画 → 逐句听朗读练听力 → 看语法讲解

## 2. 特殊场景和边界处理

**场景C：短文太短（<10字符）**
→ 提示"文章太短，至少输入10个字符"

**场景D：短文超长（>5000字符）**
→ 提示"文章过长，请限制在5000字符以内"

**场景E：内容包含敏感词**
→ 关键词匹配拦截，提示"内容包含不当词汇，请修改后重试"
→ 隐性擦边内容由AI二次审核拦截

**场景F：生成失败**
→ 3秒后自动重试一次
→ 重试仍失败则提示错误+保留已输入文本+显示手动重试按钮

**场景G：图片审核不通过**
→ 该格显示"内容审核未通过"，其余格正常显示

**场景H：手机浏览器不支持语音**
→ 自动切换为服务端音频播放，确保所有设备都能听到声音

**场景I：用户未同意协议**
→ 弹出协议弹窗挡住页面，必须同意才能使用

**场景J：无痕模式/首次访问**
→ 弹出协议弹窗（无痕模式无localStorage记录）

## 3. 用户错误操作容错

| 错误操作 | 容错机制 |
|------|------|
| 空白提交 | 提示"请输入短文" |
| 输入非英语/西语 | AI仍尝试处理，但结果可能不理想 |
| 重复点击生成 | 按钮变灰 + 加载中禁止再次点击 |
| 朗读时切换页面 | 朗读自动停止 |
| 多标签页同时打开看板 | 各标签页独立，无冲突 |
| 清除浏览器缓存 | 生词本和历史记录会丢失（弹窗已提前告知可导出） |

# 十、项目完整业务闭环总结

画词AI是一个以外语学习为核心场景的AI漫画生成工具。项目精准切入"单词记忆枯燥、阅读理解困难、听力练习断层"三大学习痛点，通过"提交短文 → AI自动拆分分镜 → 生成漫画插图 → 同步提取词汇语法 → 逐句朗读"的一站式流程，让用户在10-20秒内将任意英语或西班牙语短文转化为图文并茂的漫画教材。

产品面向中小学生、成人自学者和外语教师，支持手机和电脑浏览器直接访问。核心差异化在于"漫画视觉化学习"——相比传统纯文字查词方式，图像记忆更高效、更有趣、留存率更高。同时配套生词本、历史记录、语音朗读、数据看板等功能模块，形成"输入→学习→积累→复习→分析"的完整闭环。四道内容安全防线确保产品适合全年龄段使用。

从运营数据来看，单次生成成本约0.7元（DeepSeek文本分析几分钱 + 通义万相生图约0.6元），配合免费额度可支撑初期运营。数据看板已覆盖KPI、周月环比、用户分层和留存分析，为后续运营决策提供数据基础。

当前v1.5版本已实现完整的MVP功能闭环，但仍有AI生图质量不稳定、手机体验待优化、缺少付费体系等短板。后续迭代重点应放在成本管控、图片下载、云同步和日语扩展上，同时在运营侧推动种子用户获取（找英语/西语教师试用）和SEO优化（增加搜索关键词），逐步从工具型产品向平台型产品演进。

整体而言，画词AI是一款定位清晰、体验完整、具备差异化竞争力的AI学习工具，在AI+教育赛道有明确的成长空间和商业化潜力。
